# My Dotfiles

Desktop environment configurations for Arch Linux 

## System Overview

- **OS**: Arch Linux
- **Desktop Environments**: 
  - Hyprland (Wayland compositor)
  - KDE Plasma 6.5.5 (alternative DE)
- **Hardware**: 
  - Monitors: Triple setup (3440x1440 center, 2560x1440 vertical left, 1920x1080 right)

## Repository Structure

dotfiles/
├── hyprland/           # Hyprland window manager configuration
│   ├── hyprland.conf   # Main config (sources all others)
│   ├── monitors.conf   # Triple monitor setup
│   ├── keybinding.conf # Keyboard shortcuts
│   ├── scripts/        # Utility scripts (launcher, screenshots, etc.)
│   └── ...             # Additional config modules
├── kde/                # KDE Plasma configs (future)
├── shared/             # Shared configs (GTK, Qt themes) (future)
├── kitty/              # Terminal emulator (future)
└── waybar/             # Status bar (future)

## Installation

### Prerequisites
```bash
# Core Hyprland dependencies
sudo pacman -S hyprland kitty waybar rofi

# Utilities referenced in scripts
sudo pacman -S grim slurp wl-clipboard # Screenshot tools
sudo pacman -S nwg-displays            # Monitor configuration GUI
```

### Deploy Hyprland Configuration

1. **Backup existing config** (if any):
```bash
   mv ~/.config/hypr ~/.config/hypr.backup
```

2. **Symlink this repository**:
```bash
   ln -s ~/dotfiles/hyprland ~/.config/hypr
```

3. **Reload Hyprland**:
```bash
   hyprctl reload
```

### Environment Variables

The configuration sets the following critical environment variables:
- `HYPRSCRIPTS`: Points to `~/.config/hypr/scripts` for keybinding scripts
- Wayland/Qt/GTK variables for proper application rendering
- See `hyprland/environment.conf` for complete list
