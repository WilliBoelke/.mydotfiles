# My Dotfiles

Desktop environment configurations for Arch Linux

## System Overview

- **OS**: Arch Linux
- **Desktop**: Hyprland (Wayland compositor)
- **Hardware**:
  - Monitors: Triple setup (3440x1440 center, 2560x1440 vertical left, 1920x1080 right)

## Repository Structure
```
dotfiles/
├── hyprland/           # Hyprland window manager configuration
│   ├── hyprland.conf   # Main config (sources all others)
│   ├── monitors.conf   # Triple monitor setup
│   ├── keybinding.conf # Keyboard shortcuts
│   ├── scripts/        # Utility scripts (launcher, screenshots, etc.)
│   └── ...             # Additional config modules
├── quickshell/         # Custom Quickshell QML bar & widgets
├── config/             # App configs symlinked to ~/.config
│   ├── gtk-3.0/        # GTK3 theme settings
│   ├── gtk-4.0/        # GTK4 theme settings
│   └── qt6ct/          # Qt6 theming configuration
├── kitty/              # Terminal emulator
└── shell/              # bash & zsh configurations
```

## Theming

The **Ember** GTK theme and **Ant-Dark** icon theme are installed to `~/.local/share/` and are NOT tracked in this repo.
```bash
# Install Ember GTK theme (based on EliverLara/Ant)
git clone --depth=1 https://github.com/EliverLara/Ant.git ~/.local/share/themes/Ember
sed -i 's/Name=Ant/Name=Ember/' ~/.local/share/themes/Ember/index.theme

# Install Ant-Dark icon theme manually
# Place in ~/.local/share/icons/Ant-Dark/

# Install qt6ct color scheme placeholder
mkdir -p ~/.local/share/themes/Ember/qt6ct
cp /usr/share/qt6ct/colors/darker.conf ~/.local/share/themes/Ember/qt6ct/Ember.conf
# Note: customize Ember.conf via qt6ct GUI to match your palette
```

## Installation

### Dependencies
```bash
sudo pacman -S hyprland hyprpaper hypridle hyprlock hyprpicker \
    qt6ct qt6-svg qt6-multimedia \
    kitty grim slurp wl-clipboard cliphist \
    nwg-displays oh-my-posh quickshell \
    dolphin kasts alligator kdeconnect partitionmanager

yay -S anyrun-git
```

### Symlinks
```bash
# Hyprland
mv ~/.config/hypr ~/.config/hypr.backup
ln -s ~/dotfiles/hyprland ~/.config/hypr

# Quickshell
ln -s ~/dotfiles/quickshell ~/.config/quickshell

# Anyrun
ln -s ~/dotfiles/anyrun ~/.config/anyrun

# GTK & Qt theming
ln -s ~/dotfiles/config/gtk-3.0 ~/.config/gtk-3.0
ln -s ~/dotfiles/config/gtk-4.0 ~/.config/gtk-4.0
ln -s ~/dotfiles/config/qt6ct ~/.config/qt6ct

# Shell
ln -s ~/dotfiles/shell/.bashrc ~/.bashrc
ln -s ~/dotfiles/shell/.zshrc ~/.zshrc
ln -s ~/dotfiles/shell/bashrc ~/.config/bashrc
ln -s ~/dotfiles/shell/zshrc ~/.config/zshrc
ln -s ~/dotfiles/shell/ohmyposh ~/.config/ohmyposh
```

### Environment Variables

The configuration sets the following critical environment variables:
- `HYPRSCRIPTS`: Points to `~/.config/hypr/scripts`
- `QT_QPA_PLATFORMTHEME=qt6ct`: Qt theming via qt6ct
- Wayland/Qt/GTK variables for proper application rendering
- See `hyprland/environment.conf` for complete list
